# EminiBazar

A modern plant and gardening marketplace built with Next.js App Router, Firebase Authentication, MongoDB, and Tailwind CSS.

## Features

- Firebase Authentication
- Protected Routes
- Add Product
- Manage Products
- Update Product
- Delete Product
- Product Details Page
- Search & Filter
- MongoDB Database
- Responsive Design

## Routes

/
/about
/items
/items/[id]
/items/add
/items/manage
/items/update/[id]
/login
/register

## Technologies

- Next.js 16
- MongoDB Atlas
- Firebase Authentication
- Tailwind CSS
- Vercel

## Installation

npm install
npm run dev
