# EminiBazar

A modern plant marketplace built with Next.js and Firebase Authentication.

## Features

- Firebase Authentication
- Protected Routes
- Plant Listing
- Search Functionality
- Category Filter
- Price Filter
- Dynamic Plant Details
- Add Product
- Manage Product
- Responsive Design

## Routes

/
/items
/items/[id]
/items/add
/items/manage
/about
/login
/register

## Technologies

- Next.js App Router
- Firebase
- Tailwind CSS
